# Alert

### Alert with Link

The `<modus-alert>` utilizes the slot element, allowing you to insert links in the alert message.
Note: If anchor tags are nested, it is required to apply CSS variables `--modus-alert-<variant>-link-color` on the link, due to the limitations of nested `slot` elements.

<Story of={ComponentStories.WithLink} />

```html
<modus-alert message="Info alert (default)"></modus-alert>
<modus-alert dismissible message="Dismissible alert"></modus-alert>
<modus-alert message="Info alert with action button" button-text="Action"></modus-alert>
<modus-alert message="Error alert" type="error"></modus-alert>
<modus-alert message="Success alert" type="success"></modus-alert>
<modus-alert message="Warning alert" type="warning"></modus-alert>
<modus-alert type="info"> This is a info alert with <a href="#">an example link</a> </modus-alert>
```

### Properties

| Name              | Description                                              | Type      | Options                               | Default Value |
| ----------------- | -------------------------------------------------------- | --------- | ------------------------------------- | ------------- |
| `ariaLabel`       | The alert's aria-label                                   | `string`  |                                       |               |
| `buttonAriaLabel` | The button's aria-label                                  | `string`  |                                       |               |
| `buttonText`      | The button's text                                        | `string`  |                                       |               |
| `dismissible`     | Whether the alert is dismissible, renders the close icon | `boolean` |                                       |               |
| `message`         | The alert message                                        | `string`  |                                       |               |
| `type`            | The type of the alert                                    | `string`  | 'error', 'info', 'success', 'warning' | 'info'        |

### DOM Events

| Name           | Description                  | Type                |
| -------------- | ---------------------------- | ------------------- |
| `dismissClick` | Fires on dismiss icon click  | `CustomEvent<void>` |
| `actionClick`  | Fires on action button click | `CustomEvent<void>` |

### Accessibility

- Alert has `role` of `alert`.
- Alert gets an `aria-label` provided by the `aria-label` property input.
- When Alert has focus, <kbd>Escape</kbd> causes the `dismissClick` event to fire if dismissible.
